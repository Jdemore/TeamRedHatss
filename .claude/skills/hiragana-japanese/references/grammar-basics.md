# Hiragana Grammar Basics for Game Text

This is not a full Japanese grammar guide. It covers only what you need to write short, punchy game text in Hiragana: UI labels, tutorial prompts, dialogue barks, and flavor text.

## Sentence Structure

Japanese word order is Subject-Object-Verb (SOV), the opposite of English (SVO).

English: "The warrior strikes the beast"
Hiragana: せんし が けもの を うつ
Romaji: senshi ga kemono wo utsu
Literal: warrior [subject] beast [object] strikes

## Particles (The Glue Words)

Particles are single Hiragana characters placed after words to mark their role.

| Particle | Romaji | Role | Example |
|----------|--------|------|---------|
| は | wa | Topic marker | ぼく は つよい (boku wa tsuyoi) = "I am strong" |
| が | ga | Subject marker | りゅう が きた (ryuu ga kita) = "A dragon came" |
| を | wo | Object marker | いわ を なげる (iwa wo nageru) = "Throw a rock" |
| に | ni | Direction/target | もり に いく (mori ni iku) = "Go to the forest" |
| で | de | Location of action | ほらあな で たたかう (horaana de tatakau) = "Fight in the cave" |
| の | no | Possession | ひ の ちから (hi no chikara) = "Power of fire" |
| と | to | And / with | ひ と みず (hi to mizu) = "Fire and water" |
| も | mo | Also/too | ぼく も いく (boku mo iku) = "I'm going too" |

Note: は is normally pronounced "ha" but as a particle it is pronounced "wa". Similarly へ as a particle is "e".

## Command Form (For Game Prompts)

Games frequently use imperative/command forms for prompts and callouts.

| Dictionary Form | Command | Romaji | Meaning |
|----------------|---------|--------|---------|
| いく (iku) | いけ | ike | Go! |
| たたかう (tatakau) | たたかえ | tatakae | Fight! |
| にげる (nigeru) | にげろ | nigero | Run! / Flee! |
| うつ (utsu) | うて | ute | Strike! |
| まもる (mamoru) | まもれ | mamore | Defend! |
| みる (miru) | みろ | miro | Look! |

Softer command (request): add て (te) form
- おして (oshite) = "Press" (a button)
- えらんで (erande) = "Choose" / "Select"
- まって (matte) = "Wait"

## Adjectives

Two types, but for game UI, keep it simple:

| English | Hiragana | Romaji | Type |
|---------|----------|--------|------|
| Strong | つよい | tsuyoi | i-adjective |
| Fast | はやい | hayai | i-adjective |
| Big | おおきい | ookii | i-adjective |
| Small | ちいさい | chiisai | i-adjective |
| Difficult | むずかしい | muzukashii | i-adjective |
| Easy | かんたん | kantan | na-adjective (use かんたんな before nouns) |

## Tips for Game Text

- Short is better. Japanese game text is typically terse and punchy.
- Omit particles when space is tight -- Japanese speakers naturally drop them in casual speech and game contexts. "たたかえ!" is fine on its own without a subject.
- For UI labels, single words or two-word phrases work best: はじめ, つづき, せってい.
- Exclamation marks (!) work the same in Japanese and add urgency.
- Question marks (?) are understood but less traditional -- the particle か (ka) at the end of a sentence marks a question: もういちど か? (mou ichido ka? = "Again?")
